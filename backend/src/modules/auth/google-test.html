Google OAuth Token Generator

DEV ONLY — disabled in production
How to use:
1. Click the Sign in with Google button below
2. Sign in with any Google account
3. Copy the idToken that appears
4. Open Swagger UI (/api-docs) → POST /auth/google → paste the token

Step 1 — Sign in with Google
Active Client ID →
Step 2 — Your ID token (copy this into Swagger)

Paste into Swagger → POST /auth/google: 
{
  "idToken": "<paste token here>"
}
For a new Google user → response is needsRegistration: true.
Call again with a username to complete registration: 
{
  "idToken": "<same token>",
  "username": "your_chosen_username"
}
Tokens expire after 1 hour — refresh this page to get a new one.
